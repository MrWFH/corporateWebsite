# TechVision 企业官网

基于 Next.js 14 + TypeScript + Tailwind CSS + Framer Motion 构建的现代互联网科技企业官网。

## 技术栈

- **Next.js 14** - App Router, 静态导出
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化 CSS
- **Framer Motion** - 动画与交互
- **Lucide React** - 图标库

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000) 查看效果。

## 构建部署

```bash
npm run build
```

构建产物在 `out/` 目录，可部署到任意静态托管服务。
